Dinámicas en clases

Durante el promedio 2, realizamos distintas actividades en clases con el objetivo de implementar y presentar nuestro proyecto final del curso de Laboratorio de Innovación 2:

Dinámica 1: Resistencias

Descripción: Una resistencia es un componente eléctrico que limita la cantidad de corriente que puede fluir a través de un circuito. Es crucial en muchos circuitos electrónicos para proteger componentes sensibles.

Dinámica 2: Ultrasonido con el botón

Descripción: Este proyecto utiliza un sensor de ultrasonido para medir la distancia y un botón para activar el sensor. El resultado se mostrará en el monitor serie del Arduino.

Dinámica 3: Semáforo en Tinkercad

Descripción: Este proyecto simula un semáforo usando LEDs. Los LEDs se encenderán en secuencia para simular las luces de un semáforo: rojo, amarillo y verde.

Dinámica 4: Sensor de Ultrasonido con Servomotor

Descripción: Este proyecto utiliza un sensor de ultrasonido para detectar objetos y mover un servomotor para abrir una puerta cuando la distancia medida es menor a un umbral determinado.
